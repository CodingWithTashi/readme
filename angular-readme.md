## Installation

```
npm install -g @angular/cli
```

## Setup Project

```bash
# Clone the repository
git clone https://lptl.visualstudio.com/LuminousConnect/_git/New_Noc_Frontend

# Navigate to the project directory
cd New_Noc_Frontend

# Install dependencies
npm install

# Run project
npm run start

```
